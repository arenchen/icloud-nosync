# icloud-nosync
## Installation
```
git clone https://github.com/arenchen/icloud-nosync.git ~/icloud-nosync

chmod +x ~/icloud-nosync/icloud-nosync

export PATH="$HOME/icloud-nosync:$PATH"
```

## Usage
```
cd [project folder | workspace folder...etc]

# scan all subfolder
icloud-nosync
```
